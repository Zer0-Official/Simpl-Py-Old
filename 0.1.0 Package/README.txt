Simpl-Py is a Python 3 coded language made to create a simpler version of Python in an organized and professional format.
It may be considered as an independent language.

The Simpl-Py Application file opens Simpl-Py.
You can see the Python code in your editor with the Python file in the Extras folder.

Please Report any Bugs using the Report Bug link.
Thanks for downloading!
